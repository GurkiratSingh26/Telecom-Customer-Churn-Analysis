# Telecom Customer Churn Analysis

## Overview
This project analyzes customer churn for a telecom company to identify patterns and drivers associated with customer attrition. The goal is to translate data insights into actionable business recommendations.

## Tools Used
- Python
- pandas
- matplotlib
- seaborn
- Jupyter Notebook

## Key Questions
- What percentage of customers churn?
- Do newer customers churn more than long-term customers?
- Is churn associated with higher monthly charges?

## Key Insights
- Around 27% of customers in the dataset have churned.
- Churn is more common among customers with lower tenure.
- Customers with higher monthly charges are more likely to churn.

## Business Recommendations
- Improve onboarding and early customer experience.
- Identify high-risk customers early and apply retention strategies.
- Review pricing or value communication for higher-cost plans.
